```lua
	before = 5,

```

## Defaut key

| command | use |
| -- | -- |
| TopWindow | open a top window to show the key of csv|
| CloseWindow | close the top window|

| key | use |
| -- | -- |
|\<leader\> td| close the topwindow ,both topwindow and csv buffer has |
|\<leader\> tf| open a new topwindow |
